# 🚀 GestionTalento.TCS

## Sistema de Gestión de Talento Humano (SGTH)

**SGTH** es una aplicación web responsiva diseñada para centralizar y optimizar la gestión integral de bases de datos de talento. El objetivo principal de esta primera fase es establecer una infraestructura robusta que permita la búsqueda especializada de perfiles y la automatización del ciclo de entrevistas iniciales.

---

### ✨ Características Principales 
* **Gestión Centralizada**: Registro detallado de perfiles profesionales con captura de datos personales y expectativas salariales.
* **Motor de Búsqueda Inteligente**: Filtros dinámicos por áreas y especialidades técnicas para localizar el talento adecuado rápidamente.
* **Módulo de Entrevistas**: Herramientas para el registro de observaciones, fortalezas, debilidades y dictámenes de elegibilidad.
* **Generación de Fichas**: Automatización de la "Ficha del Candidato" en formato PDF, lista para el uso del entrevistador.
* **Administración Dinámica**: Panel de control para gestionar tablas maestras (Áreas y Especialidades) sin necesidad de modificar código.

### 🛠️ Stack Tecnológico
* **Backend**: Python con **Django** (Arquitectura relacional escalable).
* **Frontend**: **React** con Vite (Interfaz responsiva "Mobile First").
* **Base de Datos**: **PostgreSQL** (Modelo Muchos a Muchos para especialidades dinámicas).

---

> **Seguridad y Cumplimiento**: Este proyecto implementa validaciones de duplicidad por identificación y políticas de seguridad de datos (LOPD) para asegurar un manejo profesional de la información sensible.
