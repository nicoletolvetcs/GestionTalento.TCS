# GestionTalento.TCS
##🚀 Sistema de Gestión de Talento Humano (SGTH).
SGTH es una aplicación web responsiva diseñada para centralizar y optimizar la gestión integral de bases de datos de talento. El objetivo principal de esta primera fase es establecer una infraestructura robusta que permita la búsqueda especializada de perfiles y la automatización del ciclo de entrevistas iniciales.
